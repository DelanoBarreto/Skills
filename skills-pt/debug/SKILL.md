# Debug

Use esta skill para chegar na causa raiz e validar a correcao.

## Fluxo

- Reproduzir o problema.
- Isolar a menor superficie afetada.
- Ler logs, stack traces e dados de entrada.
- Formular a causa mais provavel.
- Validar com uma correcao pequena e verificavel.

## Resultado esperado

- Causa raiz clara.
- Correcao objetiva.
- Verificacao final para evitar regressao.
