# PDF Documentos

## O que faz

Ajuda em OCR, extracao de texto, conversao, organizacao e processamento em lote de arquivos PDF e documentos.

## Quando usar

- extrair dados de PDF
- OCR de documento escaneado
- converter PDF para Word
- processar lotes de arquivos
- montar pipeline documental

- `chat-with-pdf`
- `pdf-extraction`
