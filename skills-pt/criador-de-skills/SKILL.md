# Criador de Skills

## O que faz

Ajuda a criar skills novas, adaptar skills existentes para Codex, traduzir gatilhos e descricoes para portugues e organizar a biblioteca por pacotes.

## Quando usar

- criar skill nova
- adaptar skill do Claude
- adaptar skill do Gemini
- traduzir skill para portugues
- organizar aliases com `/` e `\`
- montar packs por dominio

## Base recomendada

- `skill-creator`
- `skill-installer`
- `skills`
