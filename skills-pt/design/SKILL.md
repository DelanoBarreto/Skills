# Design

Use esta skill para decidir como a interface deve parecer e se comportar.

## Foco

- Hierarquia visual.
- Fluxo de tela.
- Legibilidade e densidade de informacao.
- Consistencia do sistema visual.
- Acessibilidade e responsividade.

## Saida esperada

- Direcao visual clara.
- Decisoes justificadas.
- Regras simples para implementar sem retrabalho.
