# Designer

## O que faz

Ajuda a desenhar interfaces, telas, dashboards, paginas administrativas, hierarquia visual, componentes e experiencia de uso.

## Quando usar

- criar layout de sistema
- melhorar painel administrativo
- organizar fluxo de telas
- revisar UX e clareza visual
- montar dashboard ou pagina de gerenciamento
