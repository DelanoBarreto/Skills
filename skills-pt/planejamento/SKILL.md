# Planejamento

Use esta skill para transformar uma ideia em um plano executavel.

## O que fazer

- Entender objetivo, contexto e restricoes.
- Definir o que entra e o que nao entra.
- Quebrar o trabalho em passos ordenados.
- Explicitar riscos, dependencias e validacao.

## Como trabalhar

- Preferir plano curto, mas completo.
- Se houver ambiguidade critica, pedir a menor clarificacao possivel.
- Se o trabalho tocar varias areas, indicar quais skills complementares usar.
