**Última atualização:** Junho 2026

## 🎯 19 SKILLS PT-BR (Skills do Codex)

Use no Codex CLI ou Claude Code com: `/nome-skill descrição da tarefa`

### Frontend & Design
| Skill | Descrição |
|---|---|
| `/frontend` | Desenvolvimento de interface com React, Next.js, componentes e página. Use quando quiser criar, ajustar ou refatorar telas, componentes, estados de UI, formulários ou interações no frontend. |
| `/design` | Design de interface e experiência. Use quando quiser definir layout, visual, fluxo de telas, sistema visual, protótipo, acessibilidade ou melhoria estética. |

### Backend & APIs
| Skill | Descrição |
|---|---|
| `/backend` | Desenvolvimento de API, serviços, integrações e regra de negócio. Use quando quiser criar ou alterar rotas, validações, serviços, jobs, contratos de API ou lógica de servidor. |
| `/supabase` | Trabalho específico com Supabase (Auth, RLS, policies, SQL, RPC, storage, migrations, acesso seguro ou integração com Postgres). |
| `/banco-de-dados` | Modelagem e operação de banco de dados. Use quando precisar criar schema, migrações, índices, queries, relações, views, normalização ou ajustar persistência. |

### Testes & QA
| Skill | Descrição |
|---|---|
| `/testes` | Estratégia e implementação de testes (unitários, integração, E2E, cobertura, TDD, validação de bugfix ou desenho de cenários). |
| `/qa-automacao` | Automação de testes de navegador e validação E2E. Use quando quiser testar fluxos reais, simular usuário, validar interface, capturar regressão visual ou automatizar verificações. |

### Debugging & Troubleshooting
| Skill | Descrição |
|---|---|
| `/debug` | Investigação de erros, bugs e falhas. Use quando houver comportamento inesperado, erro em build, falha em teste, regressão, problema de runtime ou necessidade de causa raiz. |
| `/revisao` | Revisão de código e mudanças. Use quando quiser revisar PR, avaliar risco, procurar bugs, checar clareza, encontrar regressão ou receber leitura crítica do diff. |

### Segurança & Performance
| Skill | Descrição |
|---|---|
| `/seguranca` | Revisão de segurança. Use quando quiser avaliar riscos, permissão, dados sensíveis, autenticação, autorização, vulnerabilidades, exposição indevida ou boas práticas defensivas. |
| `/performance` | Análise e melhoria de performance. Use quando quiser reduzir lentidão, melhorar tempo de resposta, otimizar consultas, bundle, renderização, uso de rede ou custo operacional. |

### DevOps & CI/CD
| Skill | Descrição |
|---|---|
| `/github-ci` | Trabalho com GitHub, PR, review e CI. Use quando quiser lidar com checks, Actions, falhas de pipeline, comentários de PR, estados de review ou fluxo de integração contínua. |
| `/mcp` | Integração e uso de MCP. Use quando quiser criar, adaptar, instalar, organizar ou entender servidores MCP, ferramentas, conectores ou automatizações. |

### Arquitetura & Planejamento
| Skill | Descrição |
|---|---|
| `/arquitetura` | Arquitetura de software. Use quando precisar organizar módulos, limites, dependências, tradeoffs, padrões de estrutura, decomposição de sistema ou direção técnica. |
| `/planejamento` | Planejamento de tarefas, features e refatores. Use quando precisar decompor trabalho, definir escopo, priorizar etapas, organizar riscos ou escrever plano de execução. |

### Documentação & Roteamento
| Skill | Descrição |
|---|---|
| `/documentacao` | Documentação técnica e operacional. Use quando quiser criar ou revisar README, guias, especificações, notas de entrega, registro de decisões ou changelog. |
| `/ajuda` | Catálogo e roteamento de skills. Use quando quiser saber quais skills existem, entender o que cada uma faz ou escolher a skill certa. |
| `/roteador` | Roteamento de intenções para as skills. Use quando não souber qual skill chamar e quiser digitar o que pretende fazer. |

### Teste
| Skill | Descrição |
|---|---|
| `/teste-exemplo` | Skill de exemplo para demonstrar funcionamento do sistema. |

## 🎭 20 AGENTES ESPECIALIZADOS

Use no Claude Code com: `@nome-agente descrição da tarefa`

### Especialistas Técnicos (Desenvolvimento)

| Agente | Descrição |
|---|---|
| `@backend-specialist` | Especialista em backend para Node.js, Python e sistemas serverless/edge. Use para API development, lógica server-side, integração de banco de dados e segurança. |
| `@frontend-specialist` | Arquiteto frontend sênior em React/Next.js com mindset performance-first. Use para componentes UI, styling, state management, design responsivo ou arquitetura frontend. |
| `@database-architect` | Arquiteto de banco de dados para schema design, otimização de queries, migrações e bancos serverless. Use para operações de BD, mudanças de schema, indexação e data modeling. |
| `@mobile-developer` | Especialista em React Native e Flutter. Use para apps mobile cross-platform, features nativas e padrões mobile-específicos. |
| `@game-developer` | Desenvolvimento de games em todas as plataformas (PC, Web, Mobile, VR/AR). Use com Unity, Godot, Unreal, Phaser ou Three.js. |

### Debugging & Analysis
| Agente | Descrição |
|---|---|
| `@debugger` | Especialista em debugging sistemático, análise de causa raiz e investigação de crashes. Use para bugs complexos, issues em produção, problemas de performance ou análise de erros. |
| `@explorer-agent` | Discovery avançado de codebase e análise arquitetural profunda. Use para audits iniciais, planos de refatoring e tarefas investigativas. |
| `@code-archaeologist` | Especialista em código legado, refatoring e entendimento de sistemas não documentados. Use para ler código sujo, reverse engineering ou planejamento de modernização. |

### Testing & QA
| Agente | Descrição |
|---|---|
| `@test-engineer` | Especialista em testing, TDD e test automation. Use para escrever testes, melhorar cobertura ou debugar falhas de testes. |
| `@qa-automation-engineer` | Especialista em infraestrutura de test automation e E2E testing. Foca em Playwright, Cypress, CI pipelines e quebra de sistema. |

### Security & Compliance
| Agente | Descrição |
|---|---|
| `@security-auditor` | Especialista elite em cybersecurity. Pensa como atacante, defende como expert. OWASP 2025, supply chain security, zero trust architecture. |
| `@penetration-tester` | Especialista em offensive security, penetration testing, red team e exploitation de vulnerabilidades. Use para security assessments, attack simulations ou encontrar vulnerabilidades exploráveis. |

### DevOps & Infrastructure
| Agente | Descrição |
|---|---|
| `@devops-engineer` | Especialista em deployment, server management, CI/CD e operações em produção. CRÍTICO - Use para deployment, acesso a servidor, rollback e mudanças em produção. |
| `@performance-optimizer` | Especialista em otimização de performance, profiling, Core Web Vitals e bundle optimization. Use para melhorar speed, reduzir bundle size ou otimizar runtime. |

### Planning & Product
| Agente | Descrição |
|---|---|
| `@project-planner` | Agent inteligente de planejamento. Quebra requests em tasks, planeja estrutura de arquivos, determina quem faz o quê, cria dependency graph. |
| `@product-manager` | Especialista em product requirements, user stories e acceptance criteria. Use para definir features, esclarecer ambiguidade ou priorizar trabalho. |
| `@product-owner` | Facilitador estratégico entre necessidades de negócio e execução técnica. Especialista em elicitação de requirements, roadmap management e backlog prioritization. |

### Documentation & Coordination
| Agente | Descrição |
|---|---|
| `@documentation-writer` | Especialista em documentação técnica. Use APENAS quando explicitamente solicitar documentação (README, API docs, changelog). |
| `@seo-specialist` | Especialista em SEO e GEO (Generative Engine Optimization). Handles SEO audits, Core Web Vitals, E-E-A-T optimization e AI search visibility. |
| `@orchestrator` | Coordenação multi-agente e orquestração de tarefas. Use quando uma tarefa requerer múltiplas perspectivas, análise paralela ou execução coordenada entre domínios. |

## 💡 Dicas de Uso

### Skills PT-BR
- ✅ Use para tarefas **genéricas** e bem definidas
- ✅ Respostas em português, diretas e focadas
- ✅ Boas para **prototipagem rápida**
- ✅ Disponíveis em **Codex E Claude Code**

### Agentes Especializados
- ✅ Use para tarefas **complexas** que precisam de expertise profunda
- ✅ Melhor análise e recomendações estratégicas
- ✅ Investigação multifacetada de problemas
- ✅ Disponíveis em **Claude Code** (melhor que Codex)
