# Deploy

## O que faz

Ajuda a organizar o passo de entrega do aplicativo: build, variaveis de ambiente, validacao final e publicacao.

## Quando usar

- preparar release
- publicar nova versao
- revisar build de producao
- padronizar fluxo de entrega

## Skills de apoio

- `publicar-vps`
- `hosting`
- `infra`
- `testes`
