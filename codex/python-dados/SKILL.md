# Python Dados

## O que faz

Ajuda a criar scripts em Python para raspar dados, extrair texto de PDF, ler e gerar CSV/Excel, limpar dados e automatizar tarefas repetitivas.

## Quando usar

- extrair dados de PDF
- processar CSV ou Excel
- automatizar limpeza de dados
- raspar sites ou tabelas
- converter arquivos e montar pipelines simples

## Skills de apoio

- `banco-de-dados`
- `supabase`
- `compactar-texto`
- `caveman`
- `economia-tokens`

## Regras

- Priorize scripts pequenos e diretos.
- Preserve nomes de colunas e formatos importantes.
- Se tiver PDF escaneado, considere OCR antes da extracao.
- Se o arquivo for grande, processe em etapas.
