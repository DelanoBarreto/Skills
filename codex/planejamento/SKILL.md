# Planejamento

## O que faz

Quebra o trabalho em etapas, define prioridade, dependencias, risco tecnico e ordem de implementacao.

## Quando usar

- iniciar sistema novo
- organizar feature grande
- estruturar backlog tecnico
- planejar implementacao por fases
