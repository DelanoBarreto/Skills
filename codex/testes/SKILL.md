# Testes

## O que faz

Ajuda a definir e criar testes, validar comportamento, cobrir cenarios criticos e reduzir regressao.

## Quando usar

- criar testes unitarios
- planejar integracao
- validar fluxo de sistema
- revisar cobertura
- testar frontend e backend

## Base recomendada

- `testing-patterns`
- `webapp-testing`
- `playwright`
