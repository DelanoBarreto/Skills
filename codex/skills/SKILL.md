# Catalogo de Skills

Esta skill existe para orientar o uso das skills em portugues.

## Quando usar

Use quando o usuario:

- digitar `/skills`
- digitar `\skills`
- pedir a lista de skills
- perguntar o que cada skill faz
- quiser saber quais skills usar para juridico, financeiro, dashboard, frontend, backend, Supabase ou documentos

## Como responder

Responda sempre em portugues e em formato pratico.

Para cada skill citada, informe:

- nome curto;
- o que ela faz;
- quando vale usar;
- se faz parte da base padrao ou de um pack especifico.

## Base padrao

- `designer`: interface, layout, UX e paginas administrativas
- `frontend`: frontend web, componentes e fluxo de telas
- `backend-api`: endpoints, contratos e integracoes
- `banco-de-dados`: modelagem, tabelas, indices e migracoes
- `supabase`: composicao base para sistemas com Supabase
- `python-dados`: automacao em Python para PDF, CSV, Excel e raspagem
- `auto`: roteamento e escolha da melhor skill
- `planejamento`: quebra do trabalho em etapas
- `testes`: testes e validacao funcional
- `depuracao`: diagnostico e correcao de problemas
- `documentacao`: documentacao tecnica e funcional
- `design-sistema`: guia visual, paleta, tipografia e consistencia
- `polimento-design`: refinamento final de UI
- `hig`: padroes de interface por plataforma
- `paleta`: cor, contraste e harmonizacao visual

## Skills complementares

- `brand-guidelines`: padrao visual e identidade
- `color-expert`: cor, contraste e paletas
- `design-review`: audit visual e antes/depois
- `design-consultation`: direcao visual a partir do zero
- `frontend-design`: interface forte para web e dashboards
- `web-design-guidelines`: padroes de web UI
- `impeccable-design-polish`: acabamento final da interface
- `apple-hig`: padroes da Apple
- `shadcn-ui`: componentes e padroes do shadcn/ui
- `publicar-vps`: publicar aplicativo em VPS e hosting
- `hosting`: hospedagem, dominio e SSL
- `deploy`: entrega de aplicacao em producao
- `infra`: servidor, proxy reverso e operacao
- `economia-tokens`: resposta curta e precisa
- `resposta-curta`: encurtar a conversa sem perder o essencial
- `review-curto`: review de codigo curto e acionavel
- `commit-curto`: mensagem de commit curta e convencional
- `compactar-texto`: reduzir texto longo em arquivo ou nota
- `stats-tokens`: ver uso e economia de tokens

## Regra

Nao sugerir instalar tudo. Use primeiro a base padrao e depois os complementos de design quando fizer sentido.
