# Migracoes

## O que faz

Ajuda a planejar e revisar alteracoes de banco de dados com menor risco: tabelas, indices, constraints, versionamento e rollback.

## Quando usar

- alterar schema
- criar migracao
- revisar tabela
- migrar dados
- preparar rollback
