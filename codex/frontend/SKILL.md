# Frontend

## O que faz

Ajuda na construcao do frontend do sistema: paginas, componentes, navegacao, estados de tela e integracao com backend.

## Quando usar

- criar pagina nova
- estruturar componente React
- montar fluxo de navegacao
- integrar frontend com API
- construir area administrativa

## Base recomendada

- `frontend-design`
- `nextjs-react-expert`
- `tailwind-patterns`
- `testing-patterns`
